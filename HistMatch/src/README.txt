***For downloading this data set, we have assumed that you have the right to access the i-LIDS 
multiple-camera tracking scenario. ***


About the database:

1. Source: 2008 i-LIDS Multiple-Camera Tracking Scenario (MCTS) dataset.
2. Totally, there are 119 people and 476 person images.
3. Name of each image: The first four digits is the pedestrian id and the rest is the image index of that pedestrian.
4. Please refer to our paper when using this data set:
Ref: Wei-Shi Zheng, Shaogang Gong, and Tao Xiang, "Person Re-identification by Probabilistic Relative Distance Comparison", IEEE Conf. on CVPR, 2011.
